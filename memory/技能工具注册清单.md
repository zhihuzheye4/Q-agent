## 开发辅助脚本（dev-only，不入分发包）

> 这些脚本在开发期辅助生成代码/资源，本身不在 `q_agent` 包内，不会进 `.exe`。

### generate_icons

- **路径**：`scripts/generate_icons.py`
- **用途**：UI 矢量图生成（SVG sprite + 可选 PNG + manifest.json）
- **调用方式**：
  - `python scripts/generate_icons.py`（默认生成 SVG）
  - `python scripts/generate_icons.py --png`（按需，转 PNG 用 PySide6.QtSvg）
  - `python scripts/generate_icons.py --clean`（清理测试产物）
- **技术栈**：PySide6 + QtSvg（延迟 import，仅 `--png` 时需）
- **设计哲学**：预制调用而非实时渲染；PNG 转换按需使用；测试产物用完即删
- **状态**：已实现，5 级递进复杂度测试通过 + 6 个 UI 用图标（chat/skills/memory/new-chat/clear/about）已生成
- **当前产物**：11 个 SVG + manifest.json（含 5 个测试样本 + 6 个 UI 实用图标）

## UI 模块（q_agent/ui/）

### main_window

- **路径**：`q_agent/ui/main_window.py`
- **用途**：QMainWindow 主窗口骨架（菜单栏 + 工具栏 + 侧边栏 + QStackedWidget）
- **调用方式**：`python -m q_agent ui` 或 .exe 双击
- **设计**：活的 UI 空壳，界面可跳转、按钮可按，无实际功能
- **状态**：已实现，PySide6 6.11.1，方案 D 图标调用

### sidebar

- **路径**：`q_agent/ui/sidebar.py`
- **用途**：4 tab 切换（对话/技能/记忆/设置）
- **状态**：已实现，QListWidget + 信号驱动 QStackedWidget 切换

### pages

- **路径**：`q_agent/ui/pages/`
- **子模块**：chat_page（消息流 + 多行输入框 + 发送按钮 echo）/ skills_page（占位）/ memory_page（占位）/ settings_page（3 组控件）
- **状态**：已实现，均为活 UI 空壳，不调任何后端
- **chat_page 输入框细节**：
  - `ChatInput(QTextEdit)` 多行（取代 QLineEdit）
  - Enter 发送 / Shift+Enter / Ctrl+Enter 换行（keyPressEvent 重写）
  - `send_requested` Signal 自定义（QTextEdit 无 returnPressed）
  - 高度动态变化：监听 `documentLayout().documentSizeChanged` 信号 → 钳制 [44, 200] → `setFixedHeight`，触顶滚动条（ADR-017）
  - 气泡最大宽度 = 容器 92%（BUBBLE_WIDTH_RATIO）

### toolbar / menu_bar

- **路径**：`q_agent/ui/toolbar.py` / `menu_bar.py`
- **用途**：工具栏 3 图标按钮（新建对话/清空/关于）+ 菜单栏（文件→退出 + 帮助→关于弹窗）
- **状态**：已实现，按钮按下只显示状态消息/QMessageBox，无实际行为

### theme

- **路径**：`q_agent/ui/theme.py`
- **用途**：QSS 暗色主题（从 memory/设计规范/MASTER.md 翻译）

### icons（方案 D 核心）

- **路径**：`q_agent/ui/icons.py`
- **用途**：图标加载，QIcon 直接受 SVG + manifest.json 索引
- **方案**：D（见 ADR-016）——QIcon(svg_path)，Qt 内部 QIconEngine 按 size 智能缓存，按需渲染非全量预热
- **资源访问**：importlib.resources 兼容 PyInstaller --onefile 打包
- **状态**：已实现，11 个图标可加载
