## 工具层（Tools）— LLM 调用的底层操作

### list_models（Ollama）

- **名称**：list_models
- **描述**：查询本地 Ollama 服务可用模型列表
- **签名**：`list_models(host: str = "http://localhost:11434", timeout: float = 2.0) -> list[str]`
- **模块**：`q_agent.llm.ollama`
- **异常**：`OllamaError`（含面向用户的友好消息，覆盖连接拒绝/超时/HTTP 错误/JSON 解析失败）
- **依赖**：标准库 `urllib.request`，零新运行时依赖
- **状态**：已实现，UI 工具栏模型下拉框启动时调用
- **调用统计**：暂无

## UI 组件（不属于技能/工具，供编排层与 UI 协作参考）

### Toolbar 模型下拉框（v0.0.5 起本地/云端分组）

- **类**：`q_agent.ui.toolbar.Toolbar`
- **信号**：`model_selected(str)` — 用户切换模型时 emit 模型名
- **方法**：`refresh_models()` 触发异步检测；`current_model() -> str | None` 取当前选中
- **下拉结构**：`[本地 header（disabled）+ 本地模型/占位 + 云端 header（disabled）+ 云端预置项]`，用 `QStandardItemModel` + disabled item 做分组头
- **云端预置**：3 家代表 `gpt-4o (OpenAI) / claude-opus-4-7 (Anthropic) / gemini-2.5-pro (Google)`，未接 API（占位）
- **协作**：`ModelRefreshWorker(QThread)` 后台跑 list_models，信号 `models_found(list)` / `refresh_failed(str)` 回主线程

### ChatPage 对话页（v0.0.5 起 AI 气泡带模型名标签）

- **类**：`q_agent.ui.pages.chat_page.ChatPage`
- **方法**：`set_model_provider(callback: Callable[[], str | None])` — MainWindow 注入 `toolbar.current_model`
- **消息元组**：`(role: str, text: str, model_name: str | None)` —— AI 消息带模型名，用户消息 None
- **AI 气泡布局**：`[垂直列：模型名 QLabel#ModelLabel（11px 灰） + 气泡 QLabel#MessageAI][stretch]`

## 开发辅助脚本（dev-only，不入分发包）

> 这些脚本在开发期辅助生成代码/资源，本身不在 `q_agent` 包内，不会进 `.exe`。

### generate_icons

- **路径**：`scripts/generate_icons.py`
- **用途**：UI 矢量图生成（SVG sprite + 可选 PNG + manifest.json）
- **调用方式**：
  - `python scripts/generate_icons.py`（默认生成 SVG）
  - `python scripts/generate_icons.py --png`（按需，转 PNG 用 PySide6.QtSvg）
  - `python scripts/generate_icons.py --clean`（清理测试产物）
- **技术栈**：PySide6 + QtSvg（延迟 import，仅 `--png` 时需）
- **设计哲学**：预制调用而非实时渲染；PNG 转换按需使用；测试产物用完即删
- **状态**：已实现，5 级递进复杂度测试通过 + 6 个 UI 用图标（chat/skills/memory/new-chat/clear/about）已生成
- **当前产物**：11 个 SVG + manifest.json（含 5 个测试样本 + 6 个 UI 实用图标）

### sidebar

- **路径**：`q_agent/ui/sidebar.py`
- **用途**：4 tab 切换（对话/技能/记忆/设置）
- **状态**：已实现，QListWidget + 信号驱动 QStackedWidget 切换

### pages

- **路径**：`q_agent/ui/pages/`
- **子模块**：chat_page（消息流 + 多行输入框 + 发送按钮 echo）/ skills_page（占位）/ memory_page（占位）/ settings_page（3 组控件）
- **状态**：已实现，均为活 UI 空壳，不调任何后端
- **chat_page 输入框细节**：
  - `ChatInput(QTextEdit)` 多行（取代 QLineEdit）
  - Enter 发送 / Shift+Enter / Ctrl+Enter 换行（keyPressEvent 重写）
  - `send_requested` Signal 自定义（QTextEdit 无 returnPressed）
  - 高度动态变化：监听 `documentLayout().documentSizeChanged` 信号 → 钳制 [44, 200] → `setFixedHeight`，触顶滚动条（ADR-017）
  - 气泡最大宽度 = 容器 92%（BUBBLE_WIDTH_RATIO）

### toolbar / menu_bar

### theme

### icons（方案 D 核心）

- **路径**：`q_agent/ui/icons.py`
- **用途**：图标加载，QIcon 直接受 SVG + manifest.json 索引
- **方案**：D（见 ADR-016）——QIcon(svg_path)，Qt 内部 QIconEngine 按 size 智能缓存，按需渲染非全量预热
- **资源访问**：importlib.resources 兼容 PyInstaller --onefile 打包
- **状态**：已实现，11 个图标可加载
