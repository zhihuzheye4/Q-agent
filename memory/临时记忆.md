## 一、项目详情

- 项目名：Q-agent
- 路径：G:\agent

- 语言：Python 3.10+

## 三、上次执行

- UI 改进第一轮（v0.0.2）：
  - tooltip：theme.py 添加 `QToolTip { background: transparent; color: #F8FAFC; border: none; padding: 4px 8px; }`，文字不再被白色方块遮盖
  - 气泡宽度：chat_page.py `BUBBLE_WIDTH_RATIO` 0.7→0.92，长发言可占满容器 92% 宽度
  - 设置齿轮：generate_icons.py 换用 Feather settings 经典 8 齿 path（"M19.4 15..."），stroke-width=2 下渲染完整；重新生成 11 个 SVG
  - 输入框：chat_page.py `QLineEdit` → 自定义 `ChatInput(QTextEdit)`，多行支持长发言
    - Enter 发送 / Shift+Enter / Ctrl+Enter 换行（keyPressEvent 重写）
    - `send_requested` Signal 自定义（QTextEdit 无 returnPressed）
    - 高度区间 [44, 200]px（INPUT_MIN_HEIGHT/MAX_HEIGHT）
    - 测试更新为 setPlainText/toPlainText API
- UI 改进第二轮（v0.0.3）：
  - 输入框动态高度：监听 `QTextEdit.document().documentLayout().documentSizeChanged` 信号
  - `_adjust_height()`：doc_h + 边距 → 钳制 [44, 200] → `setFixedHeight`
  - 超过 200px 自动显示垂直滚动条，清空回 44px
  - 实测：单行 44 → 5 行 82 → 30 行触顶 200 + 滚动条 → 清空回 44
- 工具链全绿：ruff check + ruff format --check + mypy strict + pytest -q（31 测试通过，覆盖率 80.94%）
- v0.0.2 .exe 打包：`pyinstaller --onefile --windowed --add-data "q_agent/assets;q_agent/assets" q_agent/cli.py`
- v0.0.3 .exe 打包：同上，验证 Start-Process PID 22840 持续运行正常
- 安装包/README.md 版本历史表追加 v0.0.2 + v0.0.3 两行
