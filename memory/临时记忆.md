# Q-agent 临时记忆

更新时间：2026-06-20 10:35

## 一、项目详情

- 项目名：Q-agent
- 路径：G:\agent
- 仓库：https://github.com/zhihuzheye4/Q-agent
- 语言：Python 3.10+
- 定位：类似 Claude Code 的桌面端 AI 工具，纯软件形式对接本地 LLM
- 五层结构：对话/交互 → 编排/规划 → 记忆 → 技能/工具 → 工具调用层
- 关键约束：永不沙箱（仅基本输入校验）、本地 LLM 优先、零第三方依赖起步、每里程碑打 .exe、分发零外部依赖硬规则
- 当前阶段：第一个填充实际功能完成（本地 Ollama 模型检测），v0.0.4 .exe 已生成，等待用户提下一个需求
