# 方案：下拉框本地/云端分组 + 对话气泡显示模型标签

生成时间：2026-06-20 10:45
状态：待执行

## 一、需求

用户提出两个修改：
1. 下拉框区分本地和云端
2. 对话气泡上方显示当前回答的模型

## 二、决策（已与用户确认）

1. 下拉框仅 UI 分组占位——分"本地（Ollama）"和"云端"两组，云端预置常见模型名，不接云端 API
2. AI 气泡上方显示用户当前选中的模型名（echo 阶段也是显示选中名，下一阶段接真 LLM 后才是真模型）

## 三、技术设计

### 3.1 toolbar.py 改造

下拉框结构（检测成功有本地模型时）：

```
[separator/disabled header] 本地（Ollama）
qwen2.5:7b
llama3:8b
[separator/disabled header] 云端（占位，未接 API）
gpt-4o (OpenAI)
claude-opus-4-7 (Anthropic)
gemini-2.5-pro (Google)
```

实现：
- QComboBox 用 disabled 项做分组头（不可选，视觉分隔）
- `_on_models_found(models)`：
  1. clear
  2. add local header（disabled，灰色样式）
  3. add 本地模型项
  4. add cloud header（disabled）
  5. add 云端预置项（3 个：OpenAI / Anthropic / Google 各 1 个代表）
- `_on_refresh_failed`：仍显示"未发现本地 LLM"占位 + 不加云端组（让用户先解决本地问题）
- 空模型时：本地组占位"未发现本地模型"（disabled）+ 云端组照常
- `_on_combo_changed`：跳过 disabled 项（用 `itemData(index, role)` 标记 is_header）
- `current_model()`：从 itemData 检查不是 header

### 3.2 云端预置模型名

简化为 3 个代表（每家 1 个）：
- `gpt-4o (OpenAI)`
- `claude-opus-4-7 (Anthropic)`
- `gemini-2.5-pro (Google)`

后续真接 API 时可扩展为动态拉取列表（ADR-020+）。

### 3.3 chat_page.py 改造

- `_add_message(role, text, model_name: str | None = None)` 加可选参数
- AI 消息：在气泡上方加一个 QLabel 小灰字显示模型名
  - 布局改为：[垂直列：模型名 label + 气泡 label][stretch]
  - 模型名 label 用小号字体（11px）灰色（COLOR_MUTED）
  - model_name 为 None 时显示"未选模型"
- 用户消息：不变
- `_on_send_clicked()`：从 `_model_provider()` 取模型名，传给 `_add_message`

### 3.4 main_window.py 改造

- ChatPage 加 `_model_provider: Callable[[], str | None] | None = None` 属性
- MainWindow 构造后注入：`self.chat_page._model_provider = self.toolbar.current_model`
- 或者更清晰：ChatPage 加 `set_model_provider(callback)` 方法

### 3.5 测试

- test_ui_toolbar.py 补：
  - 检测成功有模型：下拉含本地 header + 模型 + 云端 header + 云端预置项
  - 检测成功无模型：本地 header + "未发现本地模型"占位 + 云端 header + 云端项
  - 检测失败：仍"未发现本地 LLM"占位
  - 切换到云端项 → emit model_selected，current_model 返回云端名
  - 切换到本地 header → 不 emit（disabled 不可选）
- test_ui_imports.py 的 test_chat_page_send_echos 更新：断言最后一条 AI 消息含模型标签

## 五、不做什么

- 不接云端 API（下一阶段）
- 不在设置页填 API key（下一阶段）
- 不在云端组动态拉取模型（预置即可）
- 用户气泡不显示模型标签（用户自己发的消息，不需要）
